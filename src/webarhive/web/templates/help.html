Возраст ≠ дата регистрации

За «возраст» принимается дата первой активности домена в архиве Wayback Machine. Это не дата регистрации домена в реестре — Wayback её не знает.

При наличии и нейтрального, и рискового контента — выбирается рисковое.

Лента эпох

Горизонтальная полоса «период → категория». Каждый блок — это эпоха, в которую тематика не менялась. Рисковые эпохи помечены красноватым цветом и иконкой (что именно). Длительные «мёртвые» промежутки сжимаются.

Редиректы и «обратить внимание»
  • Технический — внутри того же домена (например, www → корень). В карточку сворачивается под раздел.
  • Тот же сайт — то же имя, другая зона / поддомен.
  • Переезд компании — другой домен, но та же компания по контенту/бренду.
  • Обратить внимание — при сомнении уходит сюда. Всегда есть ссылка-слепок на Wayback для ручной проверки.
Предполагаемые дропы

В архиве нет прямого флага дропа. Выводим по эвристике: длинный разрыв (год+) + смена тематики после возобновления. Подаётся как предположение, не как факт. При включённом ENABLE_SMART_DROP разрывы дополнительно проверяются LLM.

Полотно без LLM-вердикта

Если выключен ENABLE_VERDICT, колонка вердикта пустая. Ориентируйтесь на флаги: рисковые иконки и «обратить внимание». Строки с флагами подсвечиваются на полотне.

LLM-провайдер и модели

В /settings выбирается провайдер (LLM_PROVIDER) и модель для каждой роли. Поддерживаются:

  • OpenRouter (по умолчанию) — единый шлюз ко множеству провайдеров. Формат модели: провайдер/модель (например openai/gpt-4o-mini, anthropic/claude-sonnet-4.5, google/gemini-2.0-flash-001). Каталог: openrouter.ai/models {{ icon("ExternalLink", 11) }}
  • OpenAI (direct) — напрямую к api.openai.com. Формат модели: голое имя (gpt-4o-mini, gpt-4o, gpt-4.1-mini, o3-mini). Каталог: platform.openai.com/docs/models {{ icon("ExternalLink", 11) }}

Тарифная сетка с ценами за 1M токенов — на странице /settings в секции «Модели по ролям». Помогает не запустить случайно дорогой прогон.

Правило большого пальца: classification вызывается до 40× на домен — экономьте (например gpt-4o-mini ~$0.15/1M). verdict вызывается 1× на домен — можно ставить умную модель.

FAQ

Прогон долго идёт. Это нормально: Internet Archive троттлит. Прогресс-бар и счётчики живые. Прерывать только если совсем встало.

Прервался посередине. На странице прогона есть кнопка «повторить» — создаст новый прогон с теми же доменами и настройками.

Домен «нет данных». Его нет в архиве Wayback. Это не ошибка — это диагноз.

Прогон с включёнными LLM-ролями встал. Скорее всего на одном из доменов очень много REVIEW-редиректов на мёртвые цели; per-domain timeout (30 мин по умолчанию) его прервёт.

Не вижу обновлённую версию после деплоя. Проверьте бейдж vX.Y в шапке справа. Если показывает старую — Docker не пересобрал образ. Перезапустите с docker compose build --no-cache webarhive.
